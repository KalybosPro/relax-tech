Slide 2/6
</> SCAFFOLDING

So I built
relax_cli

Scaffold your entire project
in a single command.

📁 Project
🧩 Feature
📄 Page
📦 Module
🗃 Model
$relaxcreatemy_app-a bloc
$relaxg featureauth
$relaxg pageauth login
$relaxg moduleproduct
$relaxg modeluser_profile

Auto-detects your architecture from pubspec.yaml.

Slide 3/6
</> FULL WORKFLOW

Beyond
scaffolding.

relax_cli covers your entire
Flutter development workflow.

# Check your environment
$relaxdoctor
# Package management
$relaxpubadd http-V ^1.2.0
# Build release artifacts
$relaxbuildapk-f production
$relaxbuildaab-f staging
# SDK version management
$relaxsdkuse3.29.0
$relaxsdkinstallstable
# Clean build artifacts
$relaxclean
