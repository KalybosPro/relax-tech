relax_image_picker
One call. Typed result.

Pick media in one line.
Customize everything.

No controllers, no plumbing. Call pick(), get a typed result split by images, videos, and documents.

attach_sheet.dart
// Open the WhatsApp-style sheetfinal result = await RelaxImagePicker.pick( context, maxSelection: 30, enableCamera: true, theme: RelaxPickerTheme(accentColor: accent),); // Typed, split by media typeresult.images // List<RelaxImageFile>result.videos // List<RelaxVideoFile>result.documents // List<RelaxDocumentFile>
